## 🚀 Quick Start

### Clone Repository

```bash
git clone https://github.com/toidicode/template.git
```

### Open Template

```bash
cd template/src/01-foodee
```

### Run Locally

Simply open:

```text
index.html
```

or use VS Code Live Server.
